# nodejs配置sop

## 一、Linux 安装 nvm（1 条命令）

[Release v0.40.4 · nvm-sh/nvm · GitHub](https://github.com/nvm-sh/nvm) 看他的readme文件，有叫我们怎么使用：

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash
```

执行完后，**重启终端** 或者 执行下面命令让它生效：

```
source ~/.bashrc
```

检查是否安装成功：

```
nvm --version
```

出现版本号 = 成功 ✅

------

## 二、用 nvm 安装 Node.js（超级简单）

1. 安装最新 LTS 版（推荐，生产稳定）

```
nvm install --lts
```

2. 安装指定版本（比如 Node 20）

```
nvm install 20
```

3. 查看当前 Node 版本

```
node -v
npm -v
```

------

## 三、常用 nvm 命令（你记这 5 个就够）

```
nvm install 20       # 安装 Node 20
nvm use 20           # 切换到 Node 20
nvm ls               # 查看已安装版本
nvm alias default 20 # 设置默认版本
nvm uninstall 18     # 卸载版本
```

这里有提到一个和nvm一样可以管理go版本的工具 gvm

**使用GVM管理多版本（高级）**

### 安装依赖

```text
yum install curl git mercurial make binutils bison gcc build-essential
```

### 安装GVM

```text
bash < <(curl -s -S -L https://raw.githubusercontent.com/moovweb/gvm/master/binscripts/gvm-installer)
```

### 生效GVM

```text
source ~/.gvm/scripts/gvm 
```

### 安装指定版本Go

```text
 gvm install go1.21.0
 gvm use go1.21.0 --default
```
